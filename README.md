# 連歌絵巻 (Renga Emaki) — RDF Triple to Haiku Renga Web Application

RDF（Resource Description Framework）のトリプルデータを深遠なる「俳句」へと読み替え、それらを右から左へと流れる絵巻物のような連歌として紡ぎ出し続ける、風流かつ実験的なウェブアプリケーションです。

---

## 🧭 コンセプト (Concept)

> **「無機質なセマンティックウェブの連なりを、日本伝統の美学である『連歌』として昇華する」**

本アプリは、`Subject (主語)`・`Predicate (述語)`・`Object (目的語)` からなるデータ構造（RDFトリプル）を、日本の定型詩である **5・7・5（上五・中七・下五）** の心象風景へと動的に変換します。

* **三部構成の読み替え:**
    * **上五（5音）:** 主語（Subject）が持つ本質や「心」を映す
    * **中七（7音）:** 述語（Predicate）が持つ関係性や「動き・働き」を表す
    * **下五（5音）:** 目的語（Object）が指し示す「結びの心」を詠む
* **連歌（れんが）としての連続性:**
    単発の俳句生成ではなく、前の句の文脈やモチーフに「付合（つけあい）」させながら、次々と句を連ねて右から左へとスクロールする絵巻物のように展開します。
* **ギリギリ読める草書体と墨絵の美:**
    Google Fonts の「Yuji Mai」を採用し、現代人が直感的に、かつ微かな謎解きのように「ギリギリ読める」流麗な草書体の縦書き表現を実現。さらに、句の内容や季節（モチーフ/季語）に応じて、HTML5 Canvas が動的に毛筆風の「墨絵」をその場で描き分け、隅に朱の落款（ハンコ）を添えます。

---

## 🛠 主な機能 (Features)

1.  **AI 宗匠による句の生成 (Online Mode):**
    アプリ内に組み込まれた Claude API が「宗匠（連歌の指導者・判定者）」となり、RDFの意味論（セマンティクス）を高い解釈力で雅な俳句へと仕立て上げます。
2.  **風雅なオフライン生成 (Offline Mode):**
    ネットワーク未接続時や API が利用できない場合でも、内蔵された「モチーフ辞書」と「語彙バンク」が作動。データ内の学術的・機械的な述語すらも、山川草木などの自然景物へと見立てた墨絵付きの句を完全スタンドアロンで自動生成し続けます。
3.  **動的・手続き的な墨絵描画 (Procedural Sumi-e):**
    句のシード値とモチーフ（山・川・海・月・鶴・魚・竹・松・梅・桜・紅葉・蓮・草・星・円相など）に基づき、淡墨の濃淡や季節の移ろいを表す色みをプログラムで1枚ずつ異なる構図で生成。
4.  **インタラクティブな読み下し:**
    草書体で書かれた文字をなぞる（またはクリックする）ことで、一時的に馴染み深い「明朝体」による読み下し表示に切り替わり、現代人でも内容を瞬時に確認できます。

## 📖 使い方 (How to Use)

### 1. 起動とお手本での体験
* 本アプリ（HTMLファイル）をブラウザで開きます。
* 画面を開くと、あらかじめ用意された「お手本（サンプル用のRDFトリプルデータ）」を用いて、すぐに自動で連歌の詠み込みが始まります。

### 2. コントロールパネルの操作
画面のUIコントロールから、絵巻の進退を自由に制御できます。
* **「詠み始める / ひと休み」ボタン:** 連歌の自動生成の開始と一時停止を切り替えます。
* **「一句詠む」ボタン:** 一時停止中に、手動で次の句を1つだけ生成させます。
* **「詠む間隔スライダー」:** 句が次々に詠まれる時間の間隔（スピード）を調整できます。
* **「お手本にもどす」ボタン:** 独自のデータを読み込んでいる状態から、最初のサンプルデータでの詠み上げに戻します。

### 3. 独自データ（.nt ファイル）の読み込み
* **「ファイルを選ぶ」ボタン** を押し、お手持ちの N-Triples 形式のデータファイル（`.nt` など）を選択します。
* **学術データとの融合:** DDBJ（日本DNAデータバンク）や UniProt などのバイオインフォマティクスデータ、各種オープンデータ（LOD）を読み込ませると、遺伝子やタンパク質の関係性といった高度に無機質な述語を、AIが風流な景物へと超解釈して詠み上げる、独特の世界観を楽しむことができます。

## 🔍 仕様と調整について

* **フォントについて:**
    本アプリでは、ウェブ上で利用可能なフォントの制約から、最も草書に近い流麗な筆致を持つ **Yuji Mai** を使用しています。より崩した表現、あるいは視認性の向上を望む場合は、CSS内の `font-family` や `letter-spacing`（字間）の数値を調整してください。
* **連歌の形式について:**
    本アプリは、ご要望の仕様に基づき **「5・7・5 の句をひたすら連ねる」** 独自の形式を採っています。もし、本格的な連歌のルール（5・7・5 の長句と 7・7 の短句を交互に組み合わせる形）へ変更したい場合は、生成エンジンのプロンプトおよびロジックを拡張することで「本式」へのカスタマイズも可能です。
